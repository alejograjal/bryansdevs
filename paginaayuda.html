Etiquetas HTML y Sus Usos

<p>

La etiqueta <p> se utiliza para crear párrafos de texto. Es una de las etiquetas más básicas en HTML.

<a>

La etiqueta <a> se usa para crear enlaces (hipervínculos) a otras páginas web o recursos en línea.

<ul> y <li>

Las etiquetas <ul> y <li> se utilizan para crear listas no ordenadas. <ul> representa la lista en sí, mientras que <li> se utiliza para cada elemento de la lista.

  • Elemento 1
  • Elemento 2
  • Elemento 3

<ol> y <li>

Las etiquetas <ol> y <li> se utilizan para crear listas ordenadas. <ol> representa la lista numerada, y <li> se utiliza para cada elemento de la lista.

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

<img>

La etiqueta <img> se usa para insertar imágenes en una página web.
